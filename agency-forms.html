✎ IT Setup — Connect Azure SSO

A Microsoft Azure admin needs to do this once:

  1. Go to portal.azure.com → Azure Active Directory → App registrations → New registration
  2. Name it "WVCAC Agency Forms", set account type to Accounts in this organizational directory only
  3. Under Redirect URI, choose Single-page application (SPA) and enter the full URL of this page (e.g. https://yourdomain.com/agency-forms.html)
  4. Click Register. Copy the Application (client) ID and Directory (tenant) ID.
  5. Open build_pages.py and find:
    const AZURE_CLIENT_ID = "";
    const AZURE_TENANT_ID = "";     Paste your IDs, then run python3 build_pages.py.

After setup: Staff sign in with their @washitavalleycac.com Microsoft account. No passwords to manage — Azure handles everything.
